# data_projects

## Overview
This repository contains multiple data analysis and machine learning projects, showcasing my ability to transform data into actionable insights. The projects focus on real-world use cases such as customer retention, HR performance, and predictive modeling, combining technical analysis with decision-making support.

---

## Projects

### 1. Databel Churn Analysis
- Analyzed customer churn behavior for a telecom company.
- Identified key drivers of churn such as contract type, customer service interactions, and usage patterns.
- Built interactive dashboards to monitor churn KPIs and support retention strategies.
- Provided actionable recommendations to reduce customer attrition.

**Techniques and Tools:**  
Power BI, DAX, Data Cleaning, KPI Analysis, Data Visualization  

---

### 2. HR Analytics Dashboard
- Developed a dashboard to analyze employee performance, attrition, and workforce trends.
- Tracked key HR metrics (turnover rate, employee distribution, performance indicators).
- Enabled data-driven HR decision-making through interactive visualizations.
- Highlighted patterns in employee retention and organizational structure.

**Techniques and Tools:**  
Power BI, Data Modeling, DAX, Data Visualization  

---

### 3. Heart Disease Risk Prediction  
- Predicts the likelihood of heart disease based on patient data.  
- Performed exploratory data analysis and preprocessing.  
- Built and evaluated machine learning models for classification.

**Techniques and Tools:**  
Python, Scikit-learn, Logistic Regression, Random Forest  

---

### 4. Iris Species Classification  
- Classifies Iris flowers into species based on sepal and petal measurements.  
- Demonstrates fundamental machine learning workflow and evaluation.

**Techniques & Tools:**  
Python, Scikit-learn, Decision Trees, Random Forest
