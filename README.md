# data_projects

## Overview
This repository contains multiple data analysis and machine learning projects, showcasing my skills in Python, Power BI, SQL, Excel, and data visualization. Each project demonstrates practical applications of data analysis, from exploratory data analysis to predictive modeling.

## Projects

1. **Heart Disease Risk Prediction**  
   - Predicts the likelihood of heart disease based on patient data.  
   - Techniques: Python, Scikit-learn, Logistic Regression, Random Forest.  
   - Includes EDA, data preprocessing, model training, and evaluation.  

2. **Iris Species Classification**  
   - Classifies Iris flowers into species based on sepal and petal measurements.  
   - Techniques: Python, Scikit-learn, Decision Trees, Random Forest.  
   - Demonstrates data preprocessing, visualization, and model performance evaluation.  

## Tools & Skills
- Languages: Python (Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn)  
- Data Visualization: Power BI, Matplotlib, Seaborn  
- Databases: SQL  
- Other: Excel, Data Cleaning, EDA, Feature Engineering, Predictive Modeling
